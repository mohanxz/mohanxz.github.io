Display : inline-block

When we are explicitly changing the display from block to inline we cannot resize the element
In that case we have to use Display : inline-block;

<h1> inline

We can not resize element when we change display from block to inline

Here h1's display property is inline

h1{
background-color: rgb(149, 123, 174);
padding: 30px;
color: black;
border-radius: 15px;
display: inline;
position: relative;
top: 50px;
The height and width will not work here.
height: 50px;
width: 500px;
}

Media Query

Media query is used for the responsiveness of the website with mobile,tab,computer.

@media screen and (max-width:500px) {
textarea{
background-color: rgb(188, 227, 149);
} }

Max-width covers from 0 to 500px

To see the change shrink the screen to less than 500px width

Min-width covers from mentioned to Infinity px

To see the change shrink the screen to greater than mentioned width
